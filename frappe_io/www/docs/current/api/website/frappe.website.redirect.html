frappe.website.redirect

Version 10.1.36 Source

frappe.website.redirect.resolve_redirect (path) 

Resolve redirects from hooks

Example:

    website_redirect = [
        # absolute location
        {"source": "/from", "target": "https://mysite/from"},

        # relative location
        {"source": "/from", "target": "/main"},

        # use regex
        {"source": "/from/(.*)", "target": "/main/?"}
    ]
